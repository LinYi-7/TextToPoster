# TextToPoster

这是一个用于沉淀“长文本转图片版图文海报”能力的仓库。

仓库核心目标是：把原文内容先提炼成结构化视觉信息，再生成 HTML 中间页，最后渲染为 PNG 图片，用于社媒传播、封面图制作、长图总结和知识卡片生成。

---

## 当前内容

目前仓库包含一份核心 skill：

- `skills/visual-summary-poster-skill.md`

这份 skill 适用于：

- 长文本总结图
- 图文海报
- 社媒长图
- 封面图
- 教程图解
- 方法论总结图
- 结构图与知识卡片

---

## Skill 能力概览

这份 skill 主要解决以下问题：

1. 如何把原文提炼成适合视觉呈现的结构
2. 如何区分“长图文”和“封面图”两种输出形态
3. 如何根据内容自动匹配风格
4. 如何支持多种图文视觉风格
5. 如何把 HTML 中间页渲染为 PNG 图片

目前内置 8 种风格：

1. 咨询卡片方法论风
2. 杂志深浅拼接风
3. 手账教程图解风
4. 极简金句分镜风
5. 小红书爆款笔记风
6. 奢华自然意境风
7. 软萌知识卡片风
8. 黑板报讲解风
